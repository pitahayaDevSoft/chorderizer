# Project Soul

## Vision
To provide composers, theorists, and music producers with a terminal-native, highly efficient harmonic workstation that removes the friction between theoretical exploration and MIDI sequence generation.

## Principles
1. **Music Theory as Code:** Algorithms should reflect correct diatonic relationships and standard musical notation.
2. **Workflow Velocity:** The TUI must be instantly responsive. The terminal is the fastest way to orchestrate ideas.
3. **Modularity:** Music theory logic (`theory_utils.py`) must remain strictly decoupled from generation (`generators.py`) and presentation (`ui.py`, `tui_app.py`).
4. **Reliability:** The output generated (MIDI files) must be structurally sound and immediately usable in any standard DAW.

## Values
- **Accuracy:** Musical math must be exact.
- **Craftsmanship:** Code should be as clean and structured as a well-composed symphony.
- **Empowerment:** Giving musicians the programmatic tools to scale their creativity.
